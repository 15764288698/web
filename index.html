MFEM is a lightweight, general, scalable C++ library for finite element methods.

Features

  • 2D and 3D, arbitrary high-order elements.
  • H1, H(curl), H(div), L2 and NURBS spaces.
  • MPI version scalable to hundreds of thousands of cores.
  • Adaptive mesh refinement (AMR): conforming, nonconforming, anisotropic.
  • Galerkin, mixed, isogeometric, DG and DPG discretizations.
  • Triangular, quadrilateral, tetrahedral and hexahedral curved elements.
  • ... and many more.

MFEM is currently used in the BLAST, GLVis and XBraid projects.
